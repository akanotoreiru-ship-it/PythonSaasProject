# isw_preparation

This module focuses on preparing textual data from ISW (Institute for the Study of War) reports using Natural Language Processing (NLP) techniques.

## Contents

- **NLP preprocessing scripts (2 files)**  
  These scripts handle the transformation of raw text into structured numerical representations suitable for machine learning.

## Functionality

The preparation pipeline includes:

- **Text cleaning** – removing noise, punctuation, and irrelevant symbols  
- **Tokenization** – splitting text into meaningful units (tokens)  
- **Normalization** – lowercasing and standardizing text  
- **Stopword removal** – filtering out common non-informative words  
- **Vectorization** – converting text into numerical features (e.g., TF-IDF or similar methods)  
- **Additional NLP techniques** – such as stemming or lemmatization  

## Purpose

The goal of this module is to convert unstructured textual ISW reports into a structured format that can be used as input features for predictive models.
