# SaaS 

Ця папка містить серверну частину (Backend) та веб-інтерфейс (Frontend) для системи прогнозування повітряних тривог. Проект використовує машинне навчання для аналізу погодних умов та звітів ISW (Institute for the Study of War) для передбачення ймовірності оголошення повітряної тривоги в різних регіонах України.

##  Основні можливості

- **Короткострокове прогнозування**: Передбачення тривог на наступні 24 години.
- **Інтеграція даних**: Автоматичний збір погодних даних через Visual Crossing API.
- **NLP Аналіз**: Парсинг та векторізація текстових звітів ISW для врахування воєнної обстановки.
- **REST API**: Гнучкі ендпоінти для отримання прогнозів по конкретних регіонах.
- **Веб-інтерфейс**: Зручна панель моніторингу з можливістю примусового оновлення даних та фільтрації.

##  Структура папки

- `app.py` — Основний Flask-додаток, що керує API та веб-сервером.
- `predict.py` — Ядро системи: завантажує модель, збирає свіжі дані та генерує `results.json`.
- `get_weather.py` — Модуль для взаємодії з погодними сервісами.
- `parse_isw.py` — Скрипт для збору та попередньої обробки звітів ISW.
- `templates/` — HTML-шаблони для веб-інтерфейсу.
- `static/` — Статичні файли: CSS-стилі та JavaScript-логіка для фронтенду.
