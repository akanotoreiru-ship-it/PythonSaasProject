# data_analysis

This folder contains Exploratory Data Analysis (EDA) of all key datasets used in the project. The goal of this section is to understand data distributions, identify patterns, detect anomalies, and support further modeling decisions.

## Contents

### a. Weather
Analysis of weather-related variables, including:
- Distribution of weather conditions
- Temporal patterns
- Correlation between features

### b. War Events (Alarms)
Exploration of air raid alarms and related events:
- Frequency and time-based patterns
- Trends across regions and time
- Identification of peak activity periods

### c. ISW Reports (TF-IDF Vectors)
Analysis of textual data from ISW reports after NLP processing:
- Examination of TF-IDF feature distributions
- Identification of important terms
- Insights into how textual signals relate to events

### d. Merged Dataset
Final combined dataset analysis:
- Correlation analysis between all features
- Detection of missing values and inconsistencies
- Validation of dataset structure and readiness for modeling

## Purpose

The purpose of this folder is to provide a comprehensive understanding of all data sources and ensure data quality before model training. It supports feature selection, hypothesis generation, and overall model improvement.
